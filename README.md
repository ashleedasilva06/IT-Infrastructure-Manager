# IT Infrastructure Manager

A full-stack system to manage and monitor IT infrastructure devices.

## Tech Stack

Frontend
- Next.js
- React
- Tailwind CSS

Backend
- FastAPI
- Python

## Features

- Device management
- Alerts system
- Maintenance tracking
- User authentication
- Activity logs
- Network monitoring

## Project Structure

backend/ → FastAPI API server  
frontend/ → Next.js frontend

## Run Backend

cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload

## Run Frontend

cd frontend
npm install
npm run dev
